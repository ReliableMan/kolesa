Допустим у тебя в верстке есть две переменных с JSON как в первом файле. Там две переменные - с параметрами фильтра и данными товаров. 

Надо:

1. Сделать простейшую верстку, в которой будет фильтр и список товаров. Можно на бутстрапе или как угодно. 

В списке товаров надо отразить для карточек:
- название
- бренд
- Ширину
- Ссылку (сделать ее вида /tyre/{Slug})

В фильтре отразить параметры фильтра в виде названия параметра и чек боксов. 

2. Сделать два компонента VUE для фильтра и для списка товаров. Когда я кликаю на ширину в фильтре, в списке должны оставаться только товары с нужной шириной. 
Общение между двумя компонентами осуществить через состояние.

нужно чтобы верстка была в html странице, а не генерировалась на лету, то есть чтобы поисковики увидели названия товаров, ссылки.
